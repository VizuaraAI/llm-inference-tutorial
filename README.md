# LLM Inference Systems — a teaching tutorial

A Wispr-Flow–styled walkthrough of building a low-latency, high-throughput LLM inference system, grounded in an ERP purchase-order chatbot.

Covers: structured output, eval harness, latency profiling, cold starts & warm pools, canary deploys, cache-aware routing, batch vs online, guardrails, disaggregated inference, and the runtime/tooling/infra design framework.
